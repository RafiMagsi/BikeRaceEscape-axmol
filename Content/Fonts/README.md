# Game Fonts

## Press Start 2P (Recommended for Arcade/Retro Games)

This game uses the **Press Start 2P** font for UI labels, providing an authentic 1980s Namco arcade aesthetic.

### Font Details
- **Name**: Press Start 2P
- **Author**: Codeman38
- **License**: Free for personal use
- **Original Source**: Based on Namco arcade games (1986 Return of Ishtar)
- **Format**: TTF (TrueType)

### Installation

To use this font, download it from any of these free sources:
1. [FFonts.net](https://webfonts.ffonts.net/Press-Start-2P-Regular.font.download)
2. [FontMirror](https://www.fontmirror.com/press-start-2p)
3. [FontSum](https://www.fontsum.com/press-start-2p-font)
4. [Font.Download](https://font.download/font/press-start-2p)

Place the TTF file (PressStart2P.ttf or PressStart2P-Regular.ttf) in this directory.

### Usage in Code

The font is used by `PZGGameFieldScene.cpp` for rendering:
- Distance label
- Coins label
- Coins add label
